## Problem
Given an array of integers `numbers` that is sorted in **non-decreasing order**.

Return the indices (**1-indexed**) of two numbers, `[index1, index2]`, such that they add up to a given target number `target` and `index1 < index2`. Note that `index1` and `index2` cannot be equal, therefore you may not use the same element twice.

There will always be **exactly one valid solution**.

Your solution must use `O(1)` additional space.

## Example
```python
Input: numbers = [1,2,3,4], target = 3

Output: [1,2]
```

## Solution
```python
class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        index1 = 0
        index2 = len(numbers) - 1

        while True:
            if numbers[index1] + numbers[index2] > target:
                index2 -= 1
            elif numbers[index1] + numbers[index2] < target:
                index1 += 1
            else:
                return [index1 + 1, index2 + 1]
```

We can setup two indices that will start from the front of the array and one that will start from the end of the array.

We can then loop indefinitely and and perform three checks.
Since we know that the input array is sorted, then we can use a Binary Search approach.

If the number at the first index plus the number at the second index if greater than the target, then we need to decrement the second index as the number at index2 is too large.

Likewise, if the number at the first index plus the number at the second index if greater than the target, then we need to decrement the second index as the number at index2 is too large.
