## Problem
Given a string `s`, return `true` if it is a **palindrome**, otherwise return `false`.

A **palindrome** is a string that reads the same forward and backward. It is also case-insensitive and ignores all non-alphanumeric characters.

**Note:** Alphanumeric characters consist of letters `(A-Z, a-z)` and numbers `(0-9)`.

## Example
```python
Input: s = "Was it a car or a cat I saw?"

Output: true
```

## Solution
```python
class Solution:
    def isPalindrome(self, s: str) -> bool:
        front = 0
        end = len(s) - 1

        while front != end and front < len(s) - 1 and end > 0:
            if not s[front].isalnum():
                front += 1
                continue
            if not s[end].isalnum():
                end -= 1
                continue
            if s[front].lower() == s[end].lower():
                front += 1
                end -= 1
            else:
                return False
        return True
```

We can use two pointers by pointing 1 to the start of the string and the other to the end of the string.

We can then loop until the front pointer equals the end pointer (the store the same integer) or the front pointer becomes greater than the length of the string - 1 or the end pointer becomes less than 0.

We can then check if the current character located at the front pointer is alphanumeric. If it is NOT, then we can advance the front pointer by 1 and skip the rest of the checking in the loop.

We can also do the same with the current character located at the end pointer.

Finally, if both characters located at the front and end pointers are the equal, the same, then we can decrement both pointers and restart the loop.

However, if they are not the same, then we can immediately return `False`
