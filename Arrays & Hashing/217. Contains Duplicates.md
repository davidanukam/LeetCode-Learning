Given an integer array `nums`, return `true` if any value appears **more than once** in the array, otherwise return `false`.

## Example
```python
Input: nums = [1, 2, 3, 3]

Output: true
```

## Solution
```python
class Solution:
    def hasDuplicate(self, nums: List[int]) -> bool:
        return len(nums) != len(set(nums))
```

Since we want to find if a list has a value more than once, then we can convert it to a set and check the sizes of the original array and the set. If their lengths are different than that means at least one number was repeated. If their lengths are the same then that means each number was unique.
