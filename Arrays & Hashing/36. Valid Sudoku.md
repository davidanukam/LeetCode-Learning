## Problem
You are given a `9 x 9` Sudoku board `board`. A Sudoku board is valid if the following rules are followed:

1. Each row must contain the digits `1-9` without duplicates.
2. Each column must contain the digits `1-9` without duplicates.
3. Each of the nine `3 x 3` sub-boxes of the grid must contain the digits `1-9` without duplicates.

Return `true` if the Sudoku board is valid, otherwise return `false`

Note: A board does not need to be full or be solvable to be valid.

## Example
```python
Input: board =
[["1","2",".",".","3",".",".",".","."],
 ["4",".",".","5",".",".",".",".","."],
 [".","9","8",".",".",".",".",".","3"],
 ["5",".",".",".","6",".",".",".","4"],
 [".",".",".","8",".","3",".",".","5"],
 ["7",".",".",".","2",".",".",".","6"],
 [".",".",".",".",".",".","2",".","."],
 [".",".",".","4","1","9",".",".","8"],
 [".",".",".",".","8",".",".","7","9"]]

Output: true
```

## Solution
```python
class Solution:
    def isValidSudoku(self, board: List[List[str]]) -> bool:
	    length = len(board)
        
        def check_for_dup(arr: List[str]):
            nums = [num for num in arr if num.isnumeric()]
            if len(nums) != len(set(nums)):
                return False
            return True
    
        # Rows
        for row in board:
            if not check_for_dup(row):
                return False
            
        # Columns
        cols = [[board[j][i] for j in range(length)] for i in range(length)]
        for col in cols:
            if not check_for_dup(col):
                return False
        
        # Sub-boxes
        for row in range(length):
            for col in range(length):
                if row in [1, 4, 7] and col in [1, 4, 7]:
                    sub_box = [
                        board[row - 1][col - 1],
                        board[row - 1][col],
                        board[row - 1][col + 1],
                        board[row][col - 1],
                        board[row][col],
                        board[row][col + 1],
                        board[row + 1][col - 1],
                        board[row + 1][col],
                        board[row + 1][col + 1],
                    ]
                    if not check_for_dup(sub_box):
                        return False
        
        return True
```

The common thing we need to check is if there are any duplicated numbers in a row/column/or sub box. So we can first get the length of the board array and create a function to see if the length of the passed in array is equal to the length of the passed in array given as a set.

Using this, we can set up three different sections we need to check. For the rows, we can check if, for each row in the board, a duplicate  number appears.

For the columns, we can create a new array that is basically a transposed version of the board array. That way we can check the columns the same way we checked the rows, as the columns are now in a row format.

Lastly, we can check the sub boxes by using offsets. We know that each board will have the same size so we can use **static** offsets to check if the surrounding boxes contain a duplicate number. We can further optimize by only using the offsets on the centre cells `[1, 4, 7]` of each sub box since the board array is `9 x 9`.
